# Dimension of matrix: number of x number of columns

|2 3|

~~~
**[Martix Elements][matrix-elements]{:target="_back"}(entries of matrix)**
~~~
# Aij = " 'i,j entry' in the ith row, jth column. "

   |12 14  -3  6|
A =| 2 18 -10 12| 
   | 1 -9 -10 -2|
   | 3  3  6   1|

A2,4 = 12
A1,3 = -3
A4,2 = 3
A5,1 = undefined (error)
~~~
