# TL;DR

- 啟用 Native UI Control: picker
   - 上面 `Project` --> `Edit Properties` --> `Capabilities` --> 點 `+` 新增
   - 搜尋 `Native UI Control` 選擇
     - 打開 `picker` 打勾 ---> Done 

- 開啟 Patch Editor
  - 上面 `View` ---> `Show/Hide Patch Editor`
  - 到 Patcher Editor 畫面 點右鍵
  - 搜尋 `Picker UI` 選擇
