### Add the Runtime: O(A + B)

```java
for (int a : arrA ) {
   print(a);
}

for (int b: arrB) {
  print(b);
}
```

### Multiply the Runtime: O(A \* B)

```java
for (int a : arrA ) {
  for (int b: arrB) {
  print(a + " , "+ b);
  }
}
```
